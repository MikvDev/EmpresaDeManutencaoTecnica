# Empresa De Manutenção Técnica

Projeto para agendar manutenções técnicas.


# Como iniciar o projeto
Use o comando:
```node main.js ```

Baixe as bibliotecas:
**Mysql2**
```npm i mysql2``` 


**Readline-sync**
``npm i readline-sync ``

## funcionalidades incluem:
- Cadastro de usuários
- Agendar manutenção
